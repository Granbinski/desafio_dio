Guia de Tags HTML

Neste guia, vamos explorar várias tags HTML úteis.

Tags de cabeçalho

As tags de cabeçalho variam de <h1> até <h6>. <h7> não existe.

Tag de parágrafo

A <p> é usada para criar um parágrafo.

Outras tags úteis

Podemos usar <small> para criar um texto pequeno, <sub> para texto subscrito e <sup> para texto sobrescrito.

Lista Ordenada e Não Ordenada

Usamos <ol> para criar uma lista ordenada e <ul> para uma lista não ordenada. Ambos utilizam <li> para os itens da lista.

  1. Primeiro item
  2. Segundo item
  • Um item
  • Outro item
Link e Citação

Podemos criar um link usando <a> e uma citação usando <blockquote>.
